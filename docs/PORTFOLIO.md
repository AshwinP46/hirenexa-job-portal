# Portfolio Assets

## Project description (long form)

**HireNexa** is a production-ready, role-based campus recruitment platform that unifies students, recruiters, and placement admins in a single workflow. It replaces fragmented spreadsheets and email threads with smart job matching, resume insights, real-time pipelines, and exportable analytics. Built on TanStack Start (React 19, SSR) with a Supabase backend, RLS-secured Postgres, Google OAuth, and a glassmorphism dark UI.
